# PWA![done](https://user-images.githubusercontent.com/62722979/174447581-1e66ade5-4809-41fe-a68d-0a8d383c5604.gif)

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm run server`  

To Start Client:

`npm run client`

To Visit App:

`localhost:3000`
